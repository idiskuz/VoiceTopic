# VoiceTopic

VoiceTopic è un Theme Component per Discourse che aggiunge una “Stanza vocale” sotto il titolo di ogni topic.

## Funzionalità
- Mostra un blocco vocale sotto il titolo del topic
- Pulsante per entrare nella stanza vocale
- UI minimal e Apple‑like
- Pronto per integrazione WebRTC

## Installazione
1. Vai in Admin → Customize → Themes → Install
2. Scegli “From a Git repository”
3. Inserisci: `https://github.com/idiskuz/VoiceTopic`
